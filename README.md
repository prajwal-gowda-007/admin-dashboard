# Admin Dashboard

A responsive admin dashboard built as part of **The Odin Project** Intermediate HTML and CSS course. This project focuses on creating complex layouts using **CSS Grid** and **Flexbox**, while practicing semantic HTML and reusable components.

## Live Demo

🔗 https://prajwal-gowda-007.github.io/admin-dashboard/

## Features

- Responsive dashboard layout using CSS Grid
- Sidebar navigation with SVG icons
- Header with search bar and user profile
- Project cards with action buttons
- Announcements section
- Trending users section
- Reusable card components
- Clean and organized HTML and CSS

## Built With

- HTML5
- CSS3
  - CSS Grid
  - Flexbox

## What I Learned

- Building complex layouts using nested CSS Grid and Flexbox
- Structuring semantic HTML for dashboard interfaces
- Creating reusable UI components
- Managing spacing, alignment, and visual hierarchy
- Using Git branches for feature development and merging changes into the main branch
- Deploying projects with GitHub Pages
